# 블렌더에서 수정한 내용을 웹 3D 로비에 적용하는 방법

블렌더(`nurion_lobby.blend`)에서 로비·캐릭터·텍스트 등을 수정한 뒤, 웹 앱에 반영하려면 **수정된 씬을 다시 glTF(.glb)로 내보내서** 프로젝트의 `nurion_lobby.glb`를 **덮어쓰기**하면 됩니다.

---

## 1. 블렌더에서 glTF(.glb)로 내보내기

1. 블렌더에서 **nurion_lobby.blend** 열기  
2. 상단 메뉴 **File → Export → glTF 2.0 (.glb/.gltf)** 선택  
3. 저장 위치를 **3D_lobby 폴더**(웹 앱이 있는 폴더)로 지정  
4. **파일 이름**을 **nurion_lobby.glb** 로 지정 (기존 파일과 동일한 이름)  
5. **Export glTF 2.0** 버튼 클릭

---

## 2. 웹 앱 폴더에 파일 교체

- 내보낸 **nurion_lobby.glb**를  
  **`c:\Users\NEW\Desktop\3D_lobby\nurion_lobby.glb`**  
  위치에 **덮어쓰기**해서 저장합니다.  
  (이미 같은 폴더에 저장했다면 그대로 덮어쓰기 된 상태입니다.)

---

## 3. 웹 앱에서 확인

1. 로컬 서버로 3D 로비 실행  
   - 예: 터미널에서 `npx serve .` 실행 후 브라우저에서 접속  
2. 브라우저에서 **새로고침(F5)**  
3. 인트로 후 로비가 **수정된 모델**로 표시되는지 확인

---

## 요약

| 단계 | 작업 |
|------|------|
| 1 | 블렌더에서 **File → Export → glTF 2.0** 로 **nurion_lobby.glb** 내보내기 |
| 2 | 내보낸 파일을 **3D_lobby** 폴더의 기존 **nurion_lobby.glb** 위에 덮어쓰기 |
| 3 | 웹 앱 페이지 **새로고침** |

매번 블렌더에서 수정할 때마다 위 과정을 반복하면, 웹 3D 로비에는 항상 최신 블렌더 씬이 적용됩니다.
